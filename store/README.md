# Chrome Web Store Assets

This directory contains assets for the Chrome Web Store listing.

## Required Assets

### Icons
- `icon-128.png` - Already in `public/icons/` (11.6KB)

### Promotional Tiles
Screenshots and promotional images should be created from the running extension.

**Required Dimensions:**
- Promotional tile: 440x280 pixels (required)
- Large promotional tile: 920x680 pixels (optional)
- Marquee promo: 1400x560 pixels (optional)

### Screenshots
Minimum 1, maximum 5 screenshots at 1280x800 or 640x400.

**Recommended screenshots:**
1. Side panel with loaded AASX showing Overview tab
2. Submodels tab with property tree expanded
3. Compliance tab showing battery passport validation
4. Compare view with two assets side-by-side
5. Documents tab with file preview
