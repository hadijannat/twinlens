## Detailed Description
```
TwinLens is a privacy-first Chrome extension for viewing Asset Administration Shell (AASX) files and Digital Product Passports.

FEATURES:
• Open AASX files via drag-and-drop or right-click menu
• View Digital Nameplate, Carbon Footprint, Technical Data
• Check compliance with EU Battery Regulation
• Compare multiple assets side-by-side
• Chat with AI about your assets (optional)
• Connect to AAS registries (BaSyx compatible)
• Scan QR codes for Digital Product Passports
• View 3D models embedded in packages (glTF/GLB)

PRIVACY:
• All processing happens locally in your browser
• No data uploaded to any servers
• AI chat uses your own API keys
• Open source

STANDARDS:
• Supports AAS Metamodel v2.0 and v3.0
• Compatible with IDTA submodel templates
• VDI 2770 document classification

PERFECT FOR:
• Manufacturing engineers reviewing digital twins
• Supply chain managers verifying product passports
• Sustainability teams auditing carbon footprints
• Developers working with Industry 4.0 standards
```

## Permission Justifications

| Permission | Justification |
|------------|---------------|
| `sidePanel` | Core functionality - TwinLens operates as a Chrome side panel to display AASX file contents and Digital Product Passport information. |
| `storage` | Stores user preferences (theme, display options) and compare cart data locally. API keys for optional AI features are stored securely in chrome.storage.local. |
| `contextMenus` | Adds "Open with TwinLens" option when right-clicking AASX files or links, providing convenient access to the viewer. |

## Single Purpose Description
TwinLens is a viewer for Asset Administration Shell (AASX) files and Digital Product Passports. It parses and displays industrial digital twin data including nameplates, technical specifications, compliance status, and embedded documents.
